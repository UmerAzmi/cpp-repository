## Snake Lite (Console C++)

A classic console-based Snake game implemented in C++.

The player controls the snake using keyboard input and tries to eat food to increase the score while avoiding collisions with walls and itself.

## Controls

* W → Up
* S → Down
* A → Left
* D → Right
* X → Exit

## Features

* Real-time movement
* Growing snake tail
* Collision detection
* Score tracking
* Console rendering

## How to Run

**Compile:**

g++ snake.cpp -o snake

**Run:**

./snake

## Concepts Used

* Arrays
* Game loop
* Real-time input
* Basic game logic
