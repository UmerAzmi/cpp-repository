## Simple Calculator (C++)

A menu-driven calculator program built using C++. It allows the user to perform basic arithmetic operations through the console.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Menu-based navigation
* Continuous usage until exit

## How to Run

**Compile:**

g++ main.cpp -o calculator

**Run:**

./calculator

## Concepts Used

* Functions
* Loops
* Conditional logic
* User input handling
