## Dodge the Blocks (C++)

A simple reflex-based console game where the player must avoid falling obstacles.

The player moves left and right to dodge incoming blocks. The score increases the longer the player survives.

## Controls

* A → Move left
* D → Move right
* X → Exit

## Features

* Real-time obstacle movement
* Collision detection
* Increasing score
* Simple console graphics

## How to Run

**Compile:**

g++ dodge.cpp -o dodge

**Run:**

./dodge

## Concepts Used

* Game loop
* Keyboard input
* Collision logic
* Random number generation
