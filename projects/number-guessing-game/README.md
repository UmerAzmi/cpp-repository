## Number Guessing Game (C++)

A simple console-based number guessing game written in C++.

The program generates a random number between 1 and 100, and the player has to guess it. After each guess, the game tells whether the guess was too high or too low until the correct number is found.

## Features

* Random number generation
* Unlimited guesses
* Attempt counter
* Simple console interaction

## How to Run

**Compile:**

g++ main.cpp -o game

**Run:**

./game

## Concepts Used

* Loops
* Conditionals
* Random number generation
* Basic input/output
