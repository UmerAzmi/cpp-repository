## Student Record Manager

A console-based student record manager that stores student names and marks using STL map.

The program allows adding students, viewing all records, and calculating the average marks.

## Features

* Add student records
* View all students
* Calculate average marks
* Menu-driven interface

## How to Run

**Compile:**

g++ main.cpp -o manager

**Run:**

./manager

## Concepts Used

* STL map
* Loops
* Functions
* Basic data management
