## How to Run

1. Ensure a C++ compiler (such as GCC or Clang) is installed.
2. Clone the repository:

```bash
git clone https://github.com/UmerAzmi/part-4.git
```

3. Navigate to the project directory:

```bash
cd part-4
```

4. Compile the desired C++ file:

```bash
g++ question1.cpp -o question1
```

5. Run the executable:

```bash
./question1
```

(On Windows, run `question1.exe`.)
