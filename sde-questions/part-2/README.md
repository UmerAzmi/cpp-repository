## How to Run

1. Ensure a C++ compiler (such as GCC or Clang) is installed.
2. Clone the repository:

```bash
git clone https://github.com/UmerAzmi/part-2.git
```

3. Navigate to the project directory:

```bash
cd part-2
```

4. Compile the program:

```bash
g++ main.cpp -o solution
```

5. Run the executable:

```bash
./solution
```

(On Windows, run `solution.exe`.)
