## How to Run

1. Ensure a C++ compiler (such as GCC or Clang) is installed.
2. Clone the repository:

```bash
git clone https://github.com/UmerAzmi/part-3.git
```

3. Navigate to the project directory:

```bash
cd part-3
```

4. Compile the desired C++ file:

```bash
g++ yourfile.cpp -o yourfile
```

5. Run the executable:

```bash
./yourfile
```

(On Windows, run `yourfile.exe`.)
